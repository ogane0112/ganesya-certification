# 借入金/貸付金

## 3-1
>問題文
```
A社に対して1000円を現金でかしつけ、借用証書を受け取った。利率は年１０％である。
```
貸付金:当社からお金を貸すこと  
借入金:当社がお金を貸すこと
> 仕分けの結果
```
貸付金 1000 | 現金 1000
```

## 3-2
>問題文
```

```

借入金:当社がお金を貸すこと

```
現金 1000 | 貸付金 1000
```

## 3-7
>問題文
```
A社に対して1000円を貸し付け,利息100円を差し引いた金額を現金で渡した。
```
貸付金:当社からお金を貸すこと  
借入金:当社がお金を貸すこと
> 仕分けの結果
```
貸付金 1000 | 現金 900
            |受取利息 100
```
省略なしの利息(補足)
```
貸付金 1000 | 現金 1000
現金    100 | 受取利息 100
```
仕分けの結果のほうの仕分けは貸方の現金-借方の現金の値を引いた値となっている！
